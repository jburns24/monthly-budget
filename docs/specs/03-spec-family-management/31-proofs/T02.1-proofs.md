## Key Implementation Details

### create_family
- Uses `select(FamilyMember).where(FamilyMember.user_id == user.id)` to check existing membership
- Raises `HTTPException(409, "User already belongs to a family")` if found
- Creates `Family(name=name, timezone=timezone, created_by=user.id)`
- Creates `FamilyMember(family_id=family.id, user_id=user.id, role="admin")`
- Flushes after each insert, returns family

### get_family_with_members
- Uses `joinedload(Family.members).joinedload(FamilyMember.user)` for eager loading
- Calls `.unique().scalar_one_or_none()` to handle joined-load deduplication
- Raises `HTTPException(404, "Family not found")` if not found

## Patterns Followed

Follows `backend/app/services/user_service.py` patterns:
- Async functions with `AsyncSession` parameter
- `select()` queries with `await db.execute()`
- `scalar_one_or_none()` for single-row queries
- `db.add()` + `await db.flush()` for inserts
- `structlog` logger via `get_logger(__name__)`
