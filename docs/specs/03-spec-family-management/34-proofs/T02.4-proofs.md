## Implementation Summary

### `require_family_member`

- Queries `FamilyMember` by `family_id` + `current_user.id`
- Returns `(user, family_member)` on success
- Raises HTTP 404 "Family not found" if user is not a member (privacy-preserving; does not reveal whether family exists)

### `require_family_admin`

- Same membership query as `require_family_member`
- Additionally checks `family_member.role == "admin"`
- Raises HTTP 404 if not a member, HTTP 403 "Admin access required" if member but not admin
- Returns `(user, family_member)` on success
