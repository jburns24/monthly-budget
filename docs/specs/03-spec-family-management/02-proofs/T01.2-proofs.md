## Tables Created

### families
- id (UUID PK), name (String 255), timezone (String 64), edit_grace_days (Integer),
  created_by (UUID FK -> users.id RESTRICT), created_at (TIMESTAMP TZ)

### family_members
- id (UUID PK), family_id (UUID FK -> families.id CASCADE), user_id (UUID FK -> users.id CASCADE),
  role (String 20), joined_at (TIMESTAMP TZ)
- CHECK: role IN ('admin', 'member')
- UNIQUE: (family_id, user_id)
- Indexes: idx_family_members_family, idx_family_members_user

### invites
- id (UUID PK), family_id (UUID FK -> families.id CASCADE), invited_user_id (UUID FK -> users.id CASCADE),
  invited_by (UUID FK -> users.id CASCADE), status (String 20), created_at (TIMESTAMP TZ), responded_at (TIMESTAMP TZ)
- CHECK: status IN ('pending', 'accepted', 'declined')
- UNIQUE: (family_id, invited_user_id, status)
- Index: idx_invites_invited_user (invited_user_id, status)
