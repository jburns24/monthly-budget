## Summary

Created `backend/app/schemas/category.py` with five Pydantic v2 schema classes:

- **CategoryCreate**: `name` (str, min_length=1, max_length=100), `icon` (str | None), `sort_order` (int = 0)
- **CategoryUpdate**: all fields optional with same constraints on `name`
- **CategoryResponse**: full read model with `model_config = ConfigDict(from_attributes=True)` for ORM hydration
- **CategoryDeleteResponse**: `message`, `deleted`, `archived` (default False), `expense_count` (default 0)
- **SeedResponse**: `message`, `created_count`

All schemas validated correctly. Ruff lint passed with no issues.
