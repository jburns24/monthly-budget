### Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/api/families/{family_id}/expenses` | require_family_member | List expenses with pagination |
| POST | `/api/families/{family_id}/expenses` | require_family_member | Create expense (status 201) |
| GET | `/api/families/{family_id}/expenses/{expense_id}` | require_family_member | Get single expense |
| PUT | `/api/families/{family_id}/expenses/{expense_id}` | require_family_member | Update expense (optimistic lock) |
| DELETE | `/api/families/{family_id}/expenses/{expense_id}` | require_family_member | Delete expense |
| GET | `/api/families/{family_id}/budget/summary` | require_family_member | Budget summary for month |

### Query Parameters
- `GET /expenses`: `year_month` (required), `category_id` (optional), `page` (default 1), `per_page` (default 50)
- `GET /budget/summary`: `month` (required, YYYY-MM format)
