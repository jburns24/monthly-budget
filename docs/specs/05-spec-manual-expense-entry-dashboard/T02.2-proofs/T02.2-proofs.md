## Summary

Implemented `get_budget_summary(db, family_id, year_month)` in `backend/app/services/expense_service.py`.

The function:
- Queries active categories for the family
- LEFT JOINs expenses filtered by family_id + year_month
- LEFT JOINs monthly_goals (via subquery) filtered by family_id + year_month
- Groups by category, computing `func.coalesce(func.sum(Expense.amount_cents), 0)` for spent_cents
- Returns `BudgetSummaryResponse` with per-category data and total_spent_cents

Also implemented helper `_compute_status(spent_cents, goal_cents)` for the green/yellow/red/none logic.
