## Key Design Decisions

- `_now_utc()` helper function extracted from `is_within_grace_period` to allow clean patching in tests
- `FamilyStub` dataclass used in unit tests to avoid SQLAlchemy instrumentation complexity
- Integration tests use `app.dependency_overrides[get_db]` to inject the test session, consistent with existing API test patterns
- Grace period check is applied in the router layer (before calling service functions) so the service remains unaware of enforcement logic
- `is_editable` field defaults to `True` in `BudgetSummaryResponse` schema for backward compatibility
