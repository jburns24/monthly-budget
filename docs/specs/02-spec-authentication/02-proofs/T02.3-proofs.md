## Changes Made

### `backend/app/services/google_oauth.py` (new)
- `exchange_code(code, code_verifier) -> str` — async POST to Google's token endpoint using `httpx.AsyncClient`; returns raw `id_token`
- `verify_id_token(id_token) -> dict` — verifies Google id_token via `google.oauth2.id_token.verify_oauth2_token` (runs in thread pool via `asyncio.to_thread`); validates `iss`, `aud`, `exp`; returns decoded payload with `sub`, `email`, `name`, `picture`

### `backend/app/services/user_service.py` (new)
- `upsert_user(google_id, email, display_name, avatar_url, db) -> tuple[User, bool]` — SELECT by `google_id`; INSERT if new (sets `created_at` + `last_login_at`); UPDATE `last_login_at`, `display_name`, `avatar_url` if existing; returns `(user, is_new_user)`

### `backend/app/config.py` (modified)
- Added `google_redirect_uri` field (default `http://localhost:5173/auth/callback`) needed by `exchange_code`

### `backend/pyproject.toml` (modified)
- Moved `httpx>=0.25.0` from dev to main dependencies (needed for production async HTTP)
- Added `requests>=2.28.0` to main dependencies (required by `google.auth.transport.requests`)
