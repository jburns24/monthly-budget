## Key Implementation Details

- **PKCE**: Uses `crypto.getRandomValues` (32 bytes → 43-char base64url code_verifier) and `crypto.subtle.digest('SHA-256')` for code_challenge
- **Storage**: code_verifier and state are stored in `sessionStorage` (by consuming components)
- **useAuth**: Fetches `/api/me` via React Query, handles 401 gracefully (returns null user), provides logout that clears the query cache
- **Types**: Strict TypeScript — User interface with id, email, display_name, avatar_url, timezone
- **RFC 7636 compliance**: SHA-256 base64url encoding with no padding (`=`), `+→-`, `/→_`
